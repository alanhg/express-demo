# express-demo

### 简介

本demo结合一些常用组件(如session/mysql等)构建一个简单的demo，方便做各种功能实验

### 功能点
1. WebShell
    - 热键支持-Ctrl F全屏切换、⌘ F终端检索，⌘ K清屏、Ctrl C执行取消
    - 终端会话录制
    - 全文检索
    - 文件编辑
2. code-server-web代理
   - 不需要开放web端口到外网，利用SSH隧道代理访问

### code-server源码拉取

```shell
git clone git@github.com:coder/code-server.git  --single-branch main

```
